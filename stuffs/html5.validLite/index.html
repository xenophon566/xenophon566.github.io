Document

            $(function(){
               $("#myform").validLite({
                  pattern: {
                    number: /^\d+$/ //validate patterns (Default: myPattern)
                  },
                  setting: {
                    getValid: '#myform', // Set validate scope (Default: body)
                    corMsgShow: true, // Show correct message (Boolean: false)
                    errMsgShow: true, // Show error message (Boolean: true)
                    corClassShow: true, // Show correct class (Boolean: false)
                    errClassShow: true, // Show error class (Boolean: true)
                    corClassName: 'correctClass', // Correct class
                    errClassName: 'errorClass', // Error class
                    corElem: 'span', // Create correct element
                    errElem: 'span', // Create error element
                    corElemClass: 'corElemClass', // correct element Class
                    errElemClass: 'errElemClass' // error element Class
                  }
               });
            });
            

            <input type="text" name="demoField"
               data-validata='{
                  "mode": "false",
                  "pattern": "require",
                  "error": {
                     "errId": "err_firstname", // message id
                     "errMsg": "Error Message", // message
                     "errClass": "errorClass" // your class name
                  },
                  "correct": {
                     "corId": "cor_firstname", // message id
                     "corMsg": "Correct Message", // message
                     "corClass": "correctClass" // your class name
                  }
               }'
            />

1. Show message Pseudocode (setting: plugin options, data: data-attributes)

            if( setting.errMsgShow is ON ) {
               if(data.errId) Show error message
            } else Do not show error message

2. Show class Pseudocode (setting: plugin options, data: data-attributes)

            if( setting.errClassShow is ON ) {
               if(data.errClass) {
                  Your classes in stylesheet
               } else if(setting.errClassName) {
                  Common class in stylesheet
               } else {
                  No error class
               }
            } else No error class

3. Show element Pseudocode (setting: plugin options, data: data-attributes)

            if( data.errId exists ) {
               show element
            } else {
               Create a new element ( defined in setting.errElem )
               Set Id to "data.errId" and Class to "setting.errElemClass"
               Insert message to HTML element
            }

Notes

1. Get a POP-up to show message when (data.errId == 'alert').

2. "setting.mode" is a anti-validate setting. Reverse the validation results.
